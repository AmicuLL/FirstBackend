---Initialize the server: npm init -y
---Install required modules: npm install express node-json-db multer
---Multer module is used for uploading images for food menu.

---To start the server, open CMD (you should be in server's directory) and call: "node ." or "node index.js"

---(AUTO)---init.bat will initialize the project and will install the necessary modules.

---(AUTO)---startServer.bat will start the server and the web page can be accesed at this link: localhost:8080 (http://localhost:8080/)
